This repository contains a **minimalist task management web application** built entirely with vanilla web technologies. Primarily for learning core web development concepts.

### Project Context

**Purpose**: Demonstrate fundamental web app development using only browser-native APIs
**Features**: Add/delete tasks, mark complete, localStorage persistence, responsive design

This project showcases how to build functional applications without external dependencies, focusing on clean code and browser capabilities.

### What It Does

- Create and delete tasks
- Mark tasks as completed
- Persist data using **localStorage**
- Maintain a clean, minimal UI

> **Note**: Zero dependencies - runs anywhere with a modern browser.

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- That's it!

### Installation

1. **Clone or download the repository**

   ```bash
   git clone [your-repo-url] basic-task-manager
   cd basic-task-manager
   ```

2. **Open in browser**

   ```bash
   # Simply open index.html
   open index.html
   # or
   npx serve .
   ```

3. **Start using immediately**
   No build step, no dependencies - just open and go!
